# 雑な説明
1. Releasesからzipファイルをダウンロード！
2. datapacksにzipファイルを入れる！(v4.0.0までは中身を展開したフォルダ内に入れたzipファイルを入れる!)
3. 導入完了メッセージが出る
4. **enjoy!**
5. あとは[wiki](../../wiki)見て細かいとこチェック！(詳しい使い方はv4.0.0公開時に書きます)
